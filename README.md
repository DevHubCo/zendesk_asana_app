# Zendesk Asana App

if you don't install `zendesk_developer_tools`, please run:
```
gem install rake
gem install zendesk_developer_tools
```

Then:
```
git clone ...
cd  ...
zat server
```

In your browser, just input `localhost:4567/iframe.html`, you can see the website.
